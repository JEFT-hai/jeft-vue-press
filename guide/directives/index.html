# vue-directive

  1. 长按事件 longTap
  2. textarea自增高 growHeight
  3. 左右滑动 slide
  4. 点击元素外面 clickOut

# longTap

v-longTap:600='longTap'
1

arg: 长按事件触发时间

# growHeight

v-grow-height="0"
1

值为额外增加高度

# slide

v-slide:50='slide'
1

返回滑动方向：left/right

# v-click-out

v-click-out='clickOut'
